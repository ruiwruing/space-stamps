# 天问·方寸 — 球形邮票展厅 · 项目说明

> 中国邮政 1951—2022 年航空航天主题邮票（24 套 · 81 枚单枚）三维球形展厅
> 最后更新：2026-07-23

---

## 一、项目是什么

一个可嵌入任意网页的 **3D 球形邮票展厅**，81 枚中国航空航天邮票经抠图 + 黑白化处理后，通过 **Fibonacci Sphere（斐波那契球）** 算法均匀分布在正球面上，整体缓慢旋转，支持拖拽交互，鼠标悬停恢复彩色，点击弹出邮票档案。

**主题**：天问·方寸 —— 于方寸之间，藏华夏问天之志。

## 三、嵌入方法

### 方式一：iframe（推荐）

把 `stamp-sphere-embed.html` 与你的网页放在同一目录：

```html
<iframe src="stamp-sphere-embed.html"
        style="width:100%; height:640px; border:0;"
        title="三维球形邮票展厅"
        loading="lazy"></iframe>
```

- 组件**自适应 iframe 尺寸**（ResizeObserver 监听容器）
- 样式、脚本**完全隔离**（`.ssr-root` 作用域 + `--ssr-` 令牌前缀），不影响宿主页面
- 档案页、加载层收在组件内部，不盖住宿主页面

### 方式二：内联

打开 `stamp-sphere-embed.html`，复制三部分到你的页面：

1. `<style>` 块（全部以 `.ssr-root` 作用域）
2. 「嵌入片段」HTML（`<div class="ssr-root" id="ssr-root">…</div>`，文件中有注释标记）
3. 页面底部三个 `<script>`（three.js + 数据 + 应用逻辑）

用 CSS 控制容器尺寸：

```css
.ssr-root { height: 640px; }   /* 默认 100vh，内联时建议覆盖 */
```

## 五、技术细节

### 5.1 图像管线

95 枚源票 → 剔除 14 张多枚图（连票/四方连/总览）→ 81 枚单枚：

1. 原始尺寸洪水削边（`killable = alpha<150 | 亮度<110`，连通到边界的区域置透明）
2. 按 alpha 裁边（pad=2）
3. LANCZOS 缩放到 480px
4. 缩放后硬化 alpha（`>96→255 else 0`）+ 3×3 闭运算补洞
5. WebP 有损编码（q80）→ base64 内联

**彩色保留**：跳过灰度化步骤，灰度/彩色切换由 GPU shader 实时完成。

### 5.2 球面分布

```
Fibonacci Sphere（等半径正球，无椭球变形）：
  y = 1 - (i / (N-1)) × 2
  r = √(1 - y²)
  θ = i × π(3 - √5)
  pos = (cos(θ)·r·R,  y·R,  sin(θ)·r·R)
  R = 3.3
```

### 5.3 Billboard Shader

每枚邮票是一个 `PlaneGeometry(1,1)` + 自定义 `ShaderMaterial`：

- **顶点着色器**：在 view space 中偏移顶点 → 始终面向相机（billboard）
- **片元着色器**：`mix(灰度, 彩色, u_colorMix)` → hover 时 `u_colorMix` 从 0 渐变到 1
- `u_opacity` 按深度分层（近 0.92 → 远 0.28）

### 5.4 动画

| 动画 | 参数 |
|------|------|
| 自动旋转 | 0.10 rad/s（reduced-motion 时为 0） |
| 拖拽惯性 | 每帧 ×0.94 衰减 |
| 浮动 | 每枚邮票独立相位 + 振幅，sin 驱动 |
| 汇聚入场 | 从随机远处 → 球面位置，ease-out cubic，逐枚延迟 16ms |
| hover 彩色 | `u_colorMix += (target - current) × 0.12` |

### 5.5 作用域隔离（嵌入版）

- 所有 CSS 选择器以 `.ssr-root` 开头
- 所有 CSS 令牌以 `--ssr-` 前缀
- 所有 DOM 查询从 `#ssr-root` 内 `querySelector`
- 指针坐标用 `getBoundingClientRect()` 容器相对计算
- 画布尺寸用 `ResizeObserver` 跟随容器
- `isolation: isolate` 独立层叠上下文

## 八、注意事项

- 组件约 **4.1 MB**（81 枚邮票 base64 内联 + three.js r158 内联），首次加载需下载完整文件
- 依赖 **Google Fonts**（Noto Serif SC / Space Mono），离线时回退系统字体
- 建议 iframe 加 `loading="lazy"`，滚动到附近再加载
- 支持 `prefers-reduced-motion`（自动关闭旋转和浮动）
- 移动端自动适配（档案页折叠为上下布局）
