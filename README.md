# 天问·方寸 — 中国航空航天邮票集展

单文件部署：将 `index.html` 上传至 GitHub Pages 即可，无任何外部依赖。

- 119 张邮票图、月面贴图、Three.js 引擎全部 base64 内联，零 CDN 依赖
- **首页 Cinematic 开场**：星场/尘流/流星 + 真 3D 月球升起 + 天问一号邮票
  重力坠落（4 次阻尼弹跳 + rotateX 摆动 + 落地挤压 + 光晕爆闪）+ 像素标题
  scramble→级联解码 + 打字机副标题/标语 + 诗文环 + 双层暗角 + 胶片颗粒
- **导航分流**：邮票集 → 序章+序厅+时间轴展厅；数字展厅 → 球形邮票展厅直达；
  航天故事/关于我们 → 敬请期待提示
- **返回首页**：序章/序厅/展厅/数字展厅四处均有返回入口，一键回到首页稳态
- **数字展厅**：81 枚航空航天邮票斐波那契球分布，拖拽旋转/滚轮推拉/悬停彩色/点击档案
- 直接双击 index.html 也可本地运行
- 强制刷新缓存：Ctrl/Cmd + Shift + R
- 支持 prefers-reduced-motion 降级（直接呈现定格画面）
